# FN-018 手書き風シェーダー

## 概要
世界の最終描画に、フルスクリーンの**手書き(スケッチ/水彩)風ポストエフェクト**を1枚かける。
シーンを一旦オフスクリーン(`sceneRT`)へ描き、スケッチ・ポストパスで画面へ出力する。

効果:
- **インクの輪郭線**(輝度の Sobel エッジ)。
- **紙の色み + 粒状ノイズ**(わら半紙のような質感)。
- **手書きの微小ゆれ(boil)**(時間を約8fpsに量子化し線を揺らす)。
- **軽い色の平坦化**(水彩のにじみ風)。

## 設計(SOLID / 安全に忘れられる)
| 層 | 要素 | 責務 |
| --- | --- | --- |
| adapter/rendering | `SketchPass` | 全画面スケッチ・ポストパス(専用シーン+全画面クアッド+ShaderMaterial)。入力テクスチャ→画面。SRP |
| adapter/rendering | `ThreeRendererAdapter` | 本描画を `sceneRT` へ→ `SketchPass` で画面へ。経過時間を time uniform へ。resize 連動 |

- 既存の描画(ポータルのレンダーターゲット群・シーン本描画)は**変更しない**。最終出力だけポスト化。
- エフェクトは `SketchPass` に閉じ込め、`ThreeRendererAdapter` は「sceneRT に描く → 渡す」だけ。
  **利用をやめれば `sceneRT.texture` を直接出すだけで従来描画に戻る**(安全に忘れられる)。
- 色管理: シーン→`sceneRT`(リニア)→スケッチ→キャンバス(sRGBエンコードは最終1回)で二重変換を避ける。
  `SketchPass` が恒等なら従来と同一の見え方になる。
- 戦闘の3Dシーン(`BattleArena3DAdapter`)は別レンダラのため本エフェクトの対象外(必要なら別途適用可能)。

## 関連コード
- `src/adapters/rendering/SketchPass.ts`(ポストパス本体・GLSL)
- `src/adapters/rendering/ThreeRendererAdapter.ts`(sceneRT 経由の最終描画・resize)

## 備考
- パラメータ(線の出やすさ・紙色・平坦化度・ゆれ量)は `SketchPass` のフラグメントシェーダ定数で調整できる。
