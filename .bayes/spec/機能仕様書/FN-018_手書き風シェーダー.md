# FN-018 手書き風シェーダー

## 概要
世界の最終描画に、フルスクリーンの**手書き(スケッチ/水彩)風ポストエフェクト**を1枚かける。
シーンを一旦オフスクリーン(`sceneRT`)へ描き、スケッチ・ポストパスで画面へ出力する。

効果:
- **インクの輪郭線**(輝度の Sobel エッジ)。
- **紙の色み + 粒状ノイズ**(わら半紙のような質感)。
- **手書きの微小ゆれ(boil)**(時間を約8fpsに量子化し線を揺らす)。
- **軽い色の平坦化**(水彩のにじみ風)。

## 設計(SOLID / 安全に忘れられる)
| 層 | 要素 | 責務 |
| --- | --- | --- |
| adapter/rendering | `SketchShader`(SketchPass.ts) | 手書き風シェーダ定義(uniforms/vertex/fragment)のデータのみ。SRP |
| adapter/rendering | `ThreeRendererAdapter` | three の `EffectComposer`(`RenderPass`→`ShaderPass(SketchShader)`)で最終出力。現在ワールドの scene を毎フレーム RenderPass に差し込み、time/resolution uniform を更新。resize 連動 |

- 既存の描画(ポータルのレンダーターゲット群・シーン本描画)は**変更しない**。最終出力だけポスト化。
- 全画面クアッド/RT管理/色管理は実績ある `EffectComposer`/`ShaderPass` に委ねる(iOS Safari でも安定)。
  **`ShaderPass` を外せば従来描画に戻る**(安全に忘れられる)。エフェクト本体はシェーダ定義データのみ。
- 戦闘の3Dシーン(`BattleArena3DAdapter`)は別レンダラのため本エフェクトの対象外(必要なら別途適用可能)。

## 関連コード
- `src/adapters/rendering/SketchPass.ts`(`SketchShader`: 手書き風シェーダ定義・GLSL)
- `src/adapters/rendering/ThreeRendererAdapter.ts`(EffectComposer/RenderPass/ShaderPass による最終描画・resize)

## 備考
- パラメータ(線の出やすさ・紙色・平坦化度・ゆれ量)は `SketchPass` のフラグメントシェーダ定数で調整できる。
