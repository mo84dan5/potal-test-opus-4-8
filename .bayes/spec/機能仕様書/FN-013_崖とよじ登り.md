# FN-013 崖とよじ登り

## 概要
各世界(昼・夜・雪・遺跡)に**崖(メサ)**を置く。崖の壁は**よじ登れる**。頂上は平らで、
踏み外すと落下し、滑空(FN-012)できる。

## 崖(メサ)
- 高さ場 `CliffField` で各世界の地形(HillyTerrain)に合成する。
- 形状: 頂上の矩形(`CLIFF.halfWidth`×`halfDepth`)が高さ `CLIFF.height(4m)`、縁から `slopeRun(0.3m)` の
  ほぼ垂直の壁。`heightAt = max(元の地形, メサ高さ)`。
- 配置: 各世界とも中心 `(16, 16)` 付近(`WorldDef.cliff`)。

## よじ登り
`MovementService.tick` の垂直解決を3分岐にする:
- **急な上り**(`床 - 現在y > FALL_STEP_THRESHOLD(0.6)`)= **よじ登り**: 1フレームの上昇を
  `CLIMB_SPEED(2.5 m/s)` に制限し、水平移動も同じ比率に抑えて壁の手前で登る(`player.climbing=true`)。
  → 高い壁は時間をかけて登る。壁はほぼ垂直なので、緩斜面のように一瞬で歩いて登れてしまうことはない。
- **段差越え**(緩い上り下り・小段差): 即スナップ(従来どおり。緩い丘・階段は影響なし)。
- **大きな落差**: 落下(FN-012)。頂上から踏み外すと落下→滑空できる。

登坂中に壁から離れる/押すのをやめると、足元に床が無ければ落下(滑空可)。画面上部に「🧗 よじ登り中」を表示。

## 後方互換
- 登坂分岐は急な上り(>0.6m/フレーム相当の段差)でのみ発動。緩斜面・階段(FN-011)・通常歩行は不変。
- 多層床(2階建て)は `floorAt` が床を抑えるため、地上で誤って崖扱いの登坂が起きない。

## テスト観点
- CliffField: 頂上=height / 斜面=線形 / 外=元地形 / 高い方を採用。
- MovementService: 急な上りはよじ登り(上昇制限・水平抑制・climbing)、押し続けると頂上到達で終了、緩斜面は即登る。
