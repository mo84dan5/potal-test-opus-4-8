# 機能仕様書 FN-009 家(屋内コンテンツ)

## 概要
昼の世界 (-10, -13) に出入りできる家がある。中には住人NPC・テレビ・テーブルがあり、左右の壁に窓がある。

## 構造

| 要素 | 内容 |
| --- | --- |
| 寸法 | 6×5m、壁高 2.6m、屋根は四角錐(れんが色) |
| ドア | 正面(+Z)中央に幅1.4m×高さ2.2mの開口(扉なし)。歩いて出入りできる |
| 窓 | 左右側面の中央に 1.2×1.0m の開口+半透明ガラス+木枠+中桟。外から中・中から外が見える |
| 床 | 木の床(高さ0.12m) |
| 地形 | 家の周辺(半径7m)は平坦化(FN-008 の平坦化スポット) |

## 当たり判定(FN-006 の円柱コライダーを流用)

- 壁: 半径0.4の円を約0.5間隔で外周に並べる(`houseWallColliderSpots`)。ドア開口だけ円を置かず、通り抜けは不可・出入りはドアのみ。
- 窓: コライダーは全高扱いのため、窓からは通れない(見えるだけ)。
- テレビ(半径0.6)・テーブル(半径0.85)にもコライダー。

## 屋内コンテンツ

| 対象 | 内容 |
| --- | --- |
| 住人NPC(静止) | テレビの方を向いて立つ(NpcSpec.yaw 指定)。吹き出し「テレビはいいぞ」、タップで家とテレビの話(3コメント) |
| テレビ | 画面にカラーバー(Canvas生成・発光)。タップでコメント2つ |
| テーブル | 円卓。接近で吹き出し「これはテーブルです」 |

## 担当レイヤ
- 定義: `src/config/worldContent.ts`(HouseSpec / HOUSE 寸法 / houseWallColliderSpots / WorldDef.house)
- 構築: `src/main.ts`(平坦化スポット・コライダー・インタラクタブル)
- 描画: `src/adapters/rendering/ThreeRendererAdapter.ts`(buildHouse / createTvScreenTexture)
