# 機能仕様書 FN-008 地形起伏

## 概要
地面になだらかな起伏(丘)があり、プレイヤー・NPCは地形に沿って上り下りしながら移動する。

## 高さ場(ドメイン: HeightField)

| 項目 | 内容 |
| --- | --- |
| 生成方法 | 正弦波の重ね合わせ `0.6·sin(0.22x)·cos(0.19z) + 0.4·sin(0.11(x+z))` × 振幅(決定的・連続・外部アセットなし) |
| 振幅(ワールド別) | 昼 0.8m / 夜 0.7m / 雪 1.2m / 遺跡 0.5m(worldContent の terrainAmplitude) |
| **平坦化スポット** | 各ポータル周辺(半径5m)とスポーン地点=原点(半径4m)は smoothstep で滑らかに高さ0へ減衰。ポータル面(y=0 前提)との整合とシームレス通過を維持 |

## 地形追従(移動)

| 対象 | 挙動 |
| --- | --- |
| プレイヤー | 毎フレーム、移動・衝突押し出しの後に足元を `h(x,z)` へスナップ。視点高は足元+1.6m で起伏に追従 |
| NPC | 徘徊の移動先で `h(x,z)` へスナップ(吹き出しアンカー・メッシュも追従) |
| 静的オブジェクト | 配置時に各座標の地形高さへ載せる(吹き出しアンカーも加算) |

## 描画

- 地面メッシュは PlaneGeometry(80×80m、64×64分割)の頂点を高さ場で変位し、法線を再計算(起伏の陰影が出る)。
- 地面模様(FN: 地面テクスチャ)は従来どおりリピート貼り。

## 担当レイヤ
- 高さ場: `src/domain/values/Terrain.ts`(HeightField / FLAT_TERRAIN / HillyTerrain)
- スナップ: `src/domain/services/MovementService.ts`(プレイヤー)、`NpcWanderService.ts`(NPC)、`TickUseCase.ts`(衝突後の再スナップ)
- 構築: `src/main.ts`(buildTerrain: 振幅+平坦化スポット)
- 描画: `src/adapters/rendering/ThreeRendererAdapter.ts`(addGround 頂点変位・オブジェクト/NPC配置)
