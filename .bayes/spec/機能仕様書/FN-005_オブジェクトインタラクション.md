# 機能仕様書 FN-005 オブジェクトインタラクション

## 概要
ワールド内のオブジェクトに近づくと吹き出しが出て、近くでタップするとメッセージウィンドウでコメントを読める。

## 接近吹き出し

| 項目 | 内容 |
| --- | --- |
| 発動条件 | 吹き出し文を持つオブジェクトの **5m以内**(水平距離)に近づく |
| 表示 | オブジェクトの頭上(アンカー高さ)をスクリーン投影した位置に吹き出し(例:「これは木です」) |
| 複数該当時 | 最も近い1つだけ表示 |
| 非表示条件 | 範囲外に出る/オブジェクトが画面外・背後になる/メッセージウィンドウ表示中 |

## タップでメッセージウィンドウ

| 項目 | 内容 |
| --- | --- |
| 発動条件 | コメント列を持つオブジェクトの **5.25m以内**(旧3.5mの1.5倍)かつ**視点の前方±60°のコーン内**でタップ(短時間 <300ms かつ 移動 <10px)。背後・真横の対象には話しかけない(至近距離は前方扱い) |
| 表示 | 画面下部にメッセージウィンドウが開き、最初のコメント(例:「これは石だ。」)を表示 |
| コメント送り | 表示中にタップするたびに次のコメントへ。最後のコメントの後のタップでウィンドウが閉じる |
| 自動終了 | 会話中に相手から **6.5m** を超えて離れるとウィンドウが自動で閉じる |
| 操作との関係 | タップはダッシュ(移動>40px)・移動・見回しと判定が重ならない。ウィンドウ表示中も移動は可能 |

## 対象オブジェクト(コンテンツ定義: src/config/worldContent.ts)

| ワールド | オブジェクト | 吹き出し | タップコメント |
| --- | --- | --- | --- |
| 昼 | 木(8本) | 「これは木です」 | なし |
| 昼 | 石(4個) | なし | 「これは石だ。」ほか3コメント |
| 夜 | クリスタル(6本) | 「これはクリスタルです」 | 「これはクリスタルだ。」ほか3コメント |
| 両方 | ポータル | 「ポータルだ。くぐると別の世界へ行ける」 | なし |

- 対象エンティティ・進行状態: `src/domain/entities/Interactable.ts` / `DialogueSession.ts`
- 近接判定: `src/domain/services/InteractionService.ts`(水平距離・範囲内最近傍)
- ユースケース: `src/application/usecases/TapInteractUseCase.ts` / `NearbyBubbleUseCase.ts`
- タップ検出: `src/adapters/input/VirtualStickInputAdapter.ts`
- スクリーン投影: `src/adapters/rendering/ThreeRendererAdapter.ts`(projectToScreen)
- UI表示(DOM): `index.html` + `src/main.ts`(updateInteractionUi)
- 配置・文言の一元管理: `src/config/worldContent.ts`(描画と判定が同じ座標を参照)
