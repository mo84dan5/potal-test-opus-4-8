# 機能仕様書 FN-004 見回し

## 概要
スワイプで視点を回転させる。左右スワイプでヨー(水平)、上下スワイプでピッチ(上下)の見回しができる。
役割は**指1本ごと**に割り当てられ(ツインスティック型):
- **最初の指**は開始ゾーンで決まる: 上半分=見回し / 下半分=移動
- **2本目の指**は位置によらず**もう一方の役割**になる(1本目が見回しなら移動、1本目が移動なら見回し)

## 操作仕様

| 操作 | 挙動 |
| --- | --- |
| **上半分**でタッチ開始(1本目) | その指は見回し役。スワイプで視点回転(パッドUIは出ない) |
| 1本目が移動中の**2本目**(位置不問) | その指は見回し役。**移動は継続したまま**スワイプで見回しできる |
| 1本目が見回し中の**2本目**(位置不問) | その指は移動役(タッチ位置に仮想パッドが出る)。**見回しを継続したまま**移動できる |
| スワイプ | 右スワイプ=右を向く(ヨー減少)、上スワイプ=上を向く(ピッチ増加) |
| 見回し役の指を離す | その指の見回しだけ終了(移動スティック役には影響しない) |
| 移動スティック役の指を離す | 即時停止(FN-002)。見回し役の指は継続 |

## 回転モデル

- 感度: スワイプ量[px] × 0.005 [rad/px]
- ゾーン判定: タッチ開始の縦位置が `y < 画面高さ/2` なら見回し、それ以外は移動(`zoneForTouch`、中央線は移動側)
- 役割決定: `roleForTouch(スティック役の有無, 他の指の有無, ゾーン)` — 指が1本もなければゾーンで決定、既に指があればスティック役の有無で補完役を割当
- 見回し役の指ごとの移動デルタを**見回し役の本数で平均化**して適用する(2本が同時に動いても2倍速にならない)
- ピッチは **±0.9 rad** でクランプ(真上・真下付近を避ける)
- ヨーは無制限(一周可能)
- ピッチはポータル通過(FN-003)の影響を受けず維持される
