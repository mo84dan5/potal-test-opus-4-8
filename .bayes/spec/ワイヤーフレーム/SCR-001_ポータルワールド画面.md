# ワイヤーフレーム SCR-001 ポータルワールド画面

```
+--------------------------------------+
| ◆ Portal Walk        [昼の世界]      | ← HUD(現在ワールド名)
|                                      |
|            ___________               |
|           /  ポータル  \              |
|          |  (向こう側の |             |
|          |   世界が見え |             |
|          |   ている)   |             |
|           \___________/              |
|                                      |
|        (3Dワールド全画面描画)         |
|     ┌────────────┐                  |
|     │これは木です │ ← 吹き出し        |
|     └─────▽──────┘  (近接時に頭上へ) |
|              ___                     |
|             / ● \   ← 仮想パッド      |
|             \___/  (タッチ中のみ表示) |
| ┌──────────────────────────────┐    |
| │ これは石だ。                  │    |
| │              タップで進む ▼   │ ← メッセージウィンドウ(タップ時)
| └──────────────────────────────┘    |
| 下半分で移動 / 上半分で見回し / はじくとダッシュ | ← 操作ヒント(数秒で消える)
+--------------------------------------+
  ※ 画面の上半分=1本指見回しゾーン
     画面の下半分=移動(仮想パッド)ゾーン
+--------------------------------------+
```

- 画面全体がWebGLキャンバス。UIはHUD(ワールド名)・仮想パッド・操作ヒントをHTMLオーバーレイで表示。
- 仮想パッドはタッチした任意の位置に出現する(固定配置ではない)。
- 縦持ち・横持ちの両方に対応(リサイズ追従)。
