# 認可制御仕様書

## 方針

本アプリは**認証・認可を持たない公開静的Webアプリ**である。

| 項目 | 内容 |
| --- | --- |
| 認証 | なし(匿名アクセス) |
| 認可 | なし(全機能を全訪問者に公開) |
| ユーザーデータ | 保持しない(サーバなし・Cookie/localStorage 不使用) |
| 配信 | GitHub Pages(public リポジトリ) |

## ロール定義

| ロール | 説明 | アクセス可能画面 |
| --- | --- | --- |
| 訪問者(全員) | URLにアクセスした全ユーザー | SCR-001 ポータルワールド画面 |
