## Issue Workerの計画

### [o] セクション0 このIssueで何をやりたいか

**要件**: シェーダーで世界を**手書き風**にする。

**設計方針(SOLID / 安全に忘れられる)**:
- 既存の描画(ポータルのレンダーターゲット群 + シーン本描画)は**変えない**。最終出力だけ、
  シーンを一旦オフスクリーン(`sceneRT`)に描き、**フルスクリーンのスケッチ・ポストパス**で
  画面へ出す(輪郭線=Sobel エッジ、紙の色み・粒状感、手書きの微小ゆれ、軽い色の平坦化)。
- ポストエフェクトは**独立クラス `SketchPass`** に閉じ込める(専用シーン+全画面квад+ShaderMaterial)。
  `ThreeRendererAdapter` は「sceneRT に描く → SketchPass に渡す」だけ(単一責務・依存は内側へ)。
  → エフェクトを止めたい時は SketchPass の利用を外して `sceneRT.texture` を直接出すだけ(安全に忘れられる)。
- 色管理: シーン→`sceneRT`(リニア)→スケッチ→キャンバス(sRGBエンコードは最終1回)で二重変換を避ける
  (SketchPass が恒等なら従来と同一の見え方)。

### [o] セクション3 実装・テスト・デプロイ

#### 方針
- 最終描画のみポスト化。既存パイプラインは不変。エフェクトは SketchPass に分離(安全に忘れられる)。

#### 結果
- SketchPass.ts: 専用シーン+全画面квад+ShaderMaterial(uniforms: tDiffuse/resolution/time)。
  render(renderer, inputTexture, time) で画面へ。setSize/dispose。
  シェーダ: Sobel エッジでインク輪郭、紙の色み+粒状ノイズ、~8fps の手書きゆれ(boil)、軽い色の平坦化。
- ThreeRendererAdapter: `sceneRT`(描画バッファサイズ)を追加。本描画を sceneRT へ→ SketchPass で画面へ。
  経過時間を加算して time uniform に。onResize で sceneRT/SketchPass をリサイズ。dispose 追加。
- 仕様: FN-018 新規、機能一覧/index 更新。
- ビルド成功・既存 vitest 全パス(描画はテスト対象外)。
- デプロイ: main push → GitHub Pages。
