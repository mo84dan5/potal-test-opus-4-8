## Issue Workerの計画

### [o] セクション0 このIssueで何をやりたいか

**要件1**: 3人称モードをデフォルトにする。
**要件2**: メッシュの面が被る箇所のテクスチャちらつき(Zファイティング)を改善する。

**Zファイティングの原因(事実確認)**:
- カメラ `near=0.05, far=300` と near が極小 → 深度バッファ精度が低く、近接面でちらつきやすい。
- **崖**: 地面メッシュ(高さ場 CliffField を頂点変位)が崖の位置で盛り上がり、その上に崖フラスタムを
  重ねるため、2つの面がほぼ同一深度になりちらつく。
- **階段**: 各段のソリッド箱が隣の段と同一平面で接し、共有面が重なってちらつく。

**方針(プレゼンテーションのみ・ドメイン非変更)**:
1. `cameraMode` の初期値を `'third'` にし、ボタン初期ラベルを「🧍 3人称」に。
2. カメラ `near` を 0.2、`far` を 200 に(深度精度を改善。プレイヤー半径的に near0.2 は安全)。
   ※ ポータルの自作シェーダが logarithmicDepthBuffer 非対応のため、対数深度は使わず near/far で改善。
3. 地面メッシュは崖(CliffField)の盛り上がりを描かず **base 地形**で描く → 崖はフラスタムのみが表現し、
   面の重なりを解消。フラスタムは底を地中へ埋めて浮きを防ぐ。
4. 階段の各段を少し重ねて配置し、同一平面の接触(共有面)を無くす。
